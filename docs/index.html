Sobre o Projeto

Uma plataforma eficiente para gerenciar suas tarefas com organização e praticidade.

O TaskHub é um sistema desenvolvido em C++ que permite gerenciar tarefas diárias de forma simples e eficaz.

Fluxo do Sistema

O sistema funciona de maneira interativa, permitindo que o usuário:

  • Adicione novas tarefas informando uma descrição.
  • Marque tarefas como concluídas pelo ID.
  • Visualize a lista de tarefas pendentes ou concluídas.
  • Encerre o programa a qualquer momento.

Principais Funcionalidades

  • Criação, edição e exclusão de tarefas
  • Filtragem e categorização de tarefas
  • Autenticação de usuários
  • Interface amigável com frontend em HTML e CSS

Uso de Programação Orientada a Objetos

O TaskHub utiliza conceitos fundamentais de POO:

  • Encapsulamento: A classe Tarefa protege seus atributos e só permite acesso controlado por métodos específicos.
  • Abstração: A classe representa tarefas do mundo real com atributos e comportamentos definidos.
  • Métodos: Funções controladas para manipular o estado das tarefas.
